User Manual

From the drop-down menu of classification types, select the type of classification you would like to use. Currently there is only the option for shape classification but this is soon to be updated!

Drag and drop your files into the box provided, or click the box to browse your device for the files. Accepted file types are all image types, .zip, .tar, .tar.gz. The box only accepts up to 10 files, but you can use one of the compressed file types to upload more than 10 images/files. If you upload a wrong file, simply click the "Remove file" link underneath to remove it.

Now, just click the “Upload and Classify!” button to get the result. To easily copy the output, simply click the “Copy results to clipboard” button, or use the "Download" button to receive a text/zip file of results.

If you run into any issues, please contact Kenji Bekki.
